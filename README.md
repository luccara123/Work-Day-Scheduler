# Work Day Scheduler
A schedule project made with HTML, CSS, JavaScript, Bootstrap, Jquery and moment.js.
It is displayed on the screen the current day, date and hour. The user can add important events to a daily planner that takes the most business hours (9am to 07pm). It is possible to save the tasks and see them later. According to the time, the background color changes, if is past, present or future.
